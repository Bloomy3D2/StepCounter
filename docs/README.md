# StepCounter - GitHub Pages

Это папка содержит файлы для GitHub Pages сайта приложения StepCounter.

## Файлы

- `index.html` - Главная страница сайта
- `privacy-policy.html` - Политика конфиденциальности
- `.nojekyll` - Отключает обработку Jekyll (для чистого HTML)

## Настройка

См. инструкцию в файле `GITHUB_PAGES_SETUP.md` в корне проекта.

## Обновление

После изменений в этой папке:

```bash
git add docs/
git commit -m "Update GitHub Pages"
git push
```
