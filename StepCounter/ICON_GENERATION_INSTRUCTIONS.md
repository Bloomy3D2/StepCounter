# Инструкция по генерации иконки приложения

## Способ 1: Использование SwiftUI Preview (Рекомендуется)

1. Откройте файл `AppIconGenerator.swift` в Xcode
2. Запустите Preview (⌘ + Option + P)
3. Сделайте скриншот иконки размером 1024x1024
4. Сохраните как PNG файл

## Способ 2: Экспорт через симулятор

1. Запустите приложение в симуляторе
2. Откройте `AppIconGenerator.swift` и запустите Preview
3. Используйте инструменты для экспорта изображения

## Способ 3: Использование онлайн-генератора

1. Создайте изображение 1024x1024 пикселей с дизайном:
   - Градиентный фон (синий → зелёный → голубой)
   - Фигура идущего человека (белая)
   - Следы под ногами
   - Текст "10K" внизу

2. Используйте онлайн-генератор иконок (например, appicon.co)

## Установка иконки в проект

1. Откройте `StepCounter/Assets.xcassets/AppIcon.appiconset/`
2. Перетащите файл `AppIcon-1024.png` (1024x1024) в Xcode
3. Xcode автоматически создаст все необходимые размеры

## Дизайн иконки

- **Фон**: Градиент от синего через зелёный к голубому
- **Фигура**: Белый силуэт идущего человека
- **Детали**: Следы под ногами, текст "10K"
- **Стиль**: Современный, минималистичный, узнаваемый
