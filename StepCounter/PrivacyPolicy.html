Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей мобильного приложения StepQuest (далее — «Приложение»).

Используя Приложение, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с условиями Политики, пожалуйста, не используйте Приложение.

3. Какие персональные данные мы собираем

Приложение собирает следующие категории персональных данных:

3.1. Данные о здоровье (HealthKit)

  • Количество шагов — для отображения статистики активности
  • Пройденная дистанция — для расчета расстояния прогулок
  • Сожженные калории — для отображения энергозатрат
  • Время активности — для статистики тренировок

Важно: Все данные о здоровье хранятся исключительно на вашем устройстве в защищенном хранилище Apple HealthKit. Мы не передаем эти данные на внешние серверы.

3.2. Данные геолокации

  • GPS-координаты — для записи и отображения маршрутов прогулок
  • Время и дата — для привязки маршрутов к конкретным прогулкам

Геолокация используется только при активной записи маршрута (функция GPS-трекинга). Данные маршрутов хранятся локально на вашем устройстве.

3.3. Данные аутентификации (Sign in with Apple)

  • User ID — уникальный идентификатор пользователя Apple
  • Email адрес — только если вы предоставили его при входе (опционально)
  • Имя — только если вы предоставили его при входе (опционально)

Данные аутентификации используются исключительно для идентификации вашего аккаунта и синхронизации данных между устройствами (если используется iCloud).

3.4. Данные устройства

  • Настройки приложения — цели шагов, предпочтения темы оформления
  • Достижения и статистика — локально сохраненные данные о прогрессе

4. Цели обработки персональных данных

Персональные данные обрабатываются в следующих целях:

  • Предоставление функциональности приложения (отображение шагов, статистики, маршрутов)
  • Создание и сохранение маршрутов прогулок
  • Отслеживание достижений и прогресса пользователя
  • Синхронизация данных между устройствами пользователя (через iCloud, если включено)
  • Улучшение работы приложения и исправление ошибок

5. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Согласия субъекта персональных данных (статья 9 Федерального закона № 152-ФЗ)
  • Необходимости для исполнения договора, стороной которого является субъект персональных данных

6. Хранение и защита данных

6.1. Локализация хранения

Все персональные данные граждан Российской Федерации хранятся и обрабатываются исключительно на территории Российской Федерации.

  • Данные о здоровье хранятся в защищенном хранилище Apple HealthKit на вашем устройстве
  • Данные маршрутов и настроек хранятся локально на вашем устройстве
  • Данные не передаются на внешние серверы, расположенные за пределами РФ

6.2. Меры защиты

  • Использование встроенных механизмов защиты iOS (Keychain, HealthKit)
  • Локальное хранение данных без передачи на внешние серверы
  • Доступ к данным только через защищенные API Apple

7. Сроки хранения данных

Персональные данные хранятся в течение:

  • Данные о здоровье: до момента удаления пользователем из HealthKit или удаления приложения
  • Данные маршрутов: до момента удаления пользователем или удаления приложения
  • Данные аутентификации: до момента выхода из аккаунта или удаления приложения
  • Настройки приложения: до момента удаления приложения

После удаления приложения все локально сохраненные данные удаляются с устройства.

8. Права субъектов персональных данных

В соответствии с Федеральным законом № 152-ФЗ, вы имеете право:

  • Получать информацию об обработке ваших персональных данных
  • Требовать уточнения персональных данных, их блокирования или уничтожения
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия оператора в уполномоченном органе по защите прав субъектов персональных данных

Для реализации ваших прав обратитесь к оператору по контактным данным, указанным в разделе 2.

9. Удаление данных

Вы можете удалить все ваши данные в любое время:

  • Через настройки приложения: раздел "Конфиденциальность" → "Удалить все данные"
  • Удалив приложение с устройства (все локальные данные будут удалены)
  • Обратившись к оператору по контактным данным

10. Передача данных третьим лицам

Мы не передаем ваши персональные данные третьим лицам.

Исключения:

  • Данные могут синхронизироваться через iCloud (если вы включили синхронизацию), что является функцией операционной системы iOS
  • Данные о здоровье хранятся в HealthKit, который является частью операционной системы iOS

11. Изменения в Политике конфиденциальности

Мы оставляем за собой право вносить изменения в настоящую Политику конфиденциальности. О существенных изменениях мы уведомим вас через приложение или по электронной почте.

Продолжая использовать Приложение после внесения изменений, вы соглашаетесь с новой версией Политики.
