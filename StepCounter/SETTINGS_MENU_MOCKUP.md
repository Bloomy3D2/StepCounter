- **iPhone 14/15 Pro Max**: Больше пространства, больше элементов в строке

### Ориентация:
- **Portrait**: Вертикальная прокрутка
- **Landscape**: Горизонтальная группировка (если нужно)

## 🎬 АНИМАЦИИ

### Появление элементов:
- **Cascade animation** - элементы появляются последовательно
- **Fade in** - плавное появление
- **Slide up** - появление снизу

### Переходы:
- **Push** - для детальных экранов
- **Sheet** - для модальных окон
- **Fade** - для алертов

### Интерактивность:
- **Scale** - при нажатии (0.95)
- **Haptic feedback** - тактильная обратная связь
- **Glow** - подсветка активных элементов
