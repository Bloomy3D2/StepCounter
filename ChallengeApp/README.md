# ChallengeApp

Приложение для челленджей с денежными ставками. Пользователи платят за вход в челлендж, выполняют условия и либо зарабатывают деньги, либо теряют их.

## Архитектура

### Экраны

1. **Splash/Entry** - Экран загрузки с логотипом и слоганом
2. **Onboarding** - 3 экрана с объяснением концепции
3. **Авторизация** - Вход через Apple, Google или без аккаунта
4. **Главный экран** - Лента доступных челленджей
5. **Детали челленджа** - Подробная информация и решение о вступлении
6. **Оплата** - Экран оплаты входа в челлендж
7. **Активный челлендж** - Основной экран для отслеживания прогресса
8. **Провал** - Экран при неудаче
9. **Победа** - Экран при успешном завершении
10. **Мой прогресс** - Статистика и история челленджей
11. **Профиль** - Настройки и управление аккаунтом

## Особенности MVP

- ✅ Все основные экраны реализованы
- ✅ Базовая логика челленджей
- ✅ Хранение данных в UserDefaults
- ✅ Навигация между экранами
- ⚠️ Платежи симулируются (для реальной интеграции нужен платежный шлюз)
- ⚠️ Авторизация симулируется (для реальной интеграции нужны SDK)

## Запуск

1. Откройте проект в Xcode
2. Выберите симулятор или устройство
3. Нажмите Run (⌘R)

## Примечания

- Приложение использует темную тему
- Все данные хранятся локально в UserDefaults
- Для тестирования создаются 3 примера челленджей
- Баланс пользователя обновляется при вступлении в челлендж и при победе
