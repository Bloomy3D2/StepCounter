# Вариант без Backend (только CloudKit)

## Что можно сделать:

### ✅ CloudKit для данных
- Хранить челленджи
- Хранить участие пользователей
- Синхронизация между устройствами
- Показывать количество участников

### ✅ StoreKit для "входа"
- Пользователь покупает "билет" на челлендж через IAP
- Apple обрабатывает платеж
- Вы получаете комиссию

### ❌ Проблемы:

1. **Деньги между пользователями**
   - IAP не позволяет переводить деньги между пользователями
   - Призовой фонд нельзя распределить автоматически
   - Нужно вручную выводить через банк

2. **Честность**
   - Пользователь может подделать данные в CloudKit
   - Нет серверной проверки выполнения
   - Легко взломать

3. **Вывод средств**
   - Apple не выводит деньги на карты пользователей
   - Вы должны вручную обрабатывать каждый вывод
   - Нарушение правил App Store (если это азартная игра)

## Вывод:
**Для MVP с виртуальными деньгами - можно.**
**Для реальных денег между пользователями - НЕТ, нужен backend.**
