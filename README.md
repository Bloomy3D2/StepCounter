# StepCounter - GitHub Pages

Этот репозиторий содержит только файлы для GitHub Pages сайта приложения StepCounter.

## Структура

- `docs/` - файлы для GitHub Pages
  - `index.html` - главная страница
  - `privacy-policy.html` - политика конфиденциальности
  - `.nojekyll` - отключает обработку Jekyll

## Сайт

GitHub Pages сайт доступен по адресу:
**https://bloomy3d2.github.io/StepCounter/**

## Исходный код

Исходный код приложения хранится в отдельном приватном репозитории.
